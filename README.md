## 🚀 Como Usar

1. **Clone o repositório:**
```bash

cd programasjavapy
```

2. **Navegue até a pasta desejada:**
```bash
cd "entrada e saida"
```

3. **Execute o programa:**
```bash

javac NomeDoArquivo.java
java NomeDoArquivo

# Para Python
python nome_do_arquivo.py
```
